# slideshow

Opis projektu
Aplikacja webowa typu slideshow umożliwiająca:
- przeglądanie zdjęć za pomocą strzałek
- zarządzanie zdjęciami (dodawanie, usuwanie, zmiana nazw)
- podgląd zdjęć w czasie rzeczywistym

Jak uruchomić projekt
1. Pobierz repozytorium:
   git clone ...

2. Przenieś pliki do folderu htdocs (XAMPP)

3. Uruchom XAMPP i włącz:
   - Apache
   - MySQL

4. Otwórz phpMyAdmin i zaimportuj plik:
   slideshow-2.sql

Struktura projektu
- slideshow.php – główna strona slideshow
- panel.php – panel administracyjny
- obraz.php – obsługa zdjęć
- img1.png, img2.png... – przykładowe zdjęcia
- slideshow-2.sql – baza danych

Technologie
- PHP
- MySQL
- HTML
- CSS
